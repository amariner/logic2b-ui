## Usage

```tsx
import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger,
} from "@/components/ui/accordion"

<Accordion type="single" collapsible>
  <AccordionItem value="item-1">
    <AccordionTrigger>Is it accessible?</AccordionTrigger>
    <AccordionContent>
      Yes. It adheres to the WAI-ARIA design pattern.
    </AccordionContent>
  </AccordionItem>
</Accordion>
```

## API

| Prop | Type | Default |
| --- | --- | --- |
| `type` (Accordion) | `single \| multiple` | — |
| `collapsible` (Accordion) | `boolean` — allow closing the open item | `false` |
| `value` (AccordionItem) | `string` | — |
