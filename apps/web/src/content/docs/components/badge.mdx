## Installation

```bash
npx shadcn@latest add https://ui.logic2b.dev/r/badge.json
```

## Usage

```tsx
import { Badge } from "@/components/ui/badge"

<Badge variant="outline">Badge</Badge>
```

Use `asChild` to render a link that looks like a badge.
