## Usage

```tsx
import {
  ContextMenu,
  ContextMenuContent,
  ContextMenuItem,
  ContextMenuTrigger,
} from "@/components/ui/context-menu"

<ContextMenu>
  <ContextMenuTrigger>Right click here</ContextMenuTrigger>
  <ContextMenuContent>
    <ContextMenuItem>Back</ContextMenuItem>
    <ContextMenuItem>Reload</ContextMenuItem>
  </ContextMenuContent>
</ContextMenu>
```

## API

| Prop | Type | Default |
| --- | --- | --- |
| `variant` (ContextMenuItem) | `default \| destructive` | `default` |
| `inset` (ContextMenuItem / Label / SubTrigger) | `boolean` | `false` |
| `checked` (ContextMenuCheckboxItem) | `boolean` | — |
