## Installation

```bash
npx logic2b@latest add hover-card
```

## Usage

```tsx
import {
  HoverCard,
  HoverCardContent,
  HoverCardTrigger,
} from "@/components/ui/hover-card"

<HoverCard>
  <HoverCardTrigger>@logic2b</HoverCardTrigger>
  <HoverCardContent>
    Copy-paste components for building beautiful interfaces, fast.
  </HoverCardContent>
</HoverCard>
```

## API

| Prop | Type | Default |
| --- | --- | --- |
| `align` (HoverCardContent) | `start \| center \| end` | `center` |
| `sideOffset` (HoverCardContent) | `number` | `4` |
| `openDelay` / `closeDelay` (HoverCard) | `number` (ms) | `700` / `300` |
