## Installation

```bash
npx shadcn@latest add https://ui.logic2b.dev/r/input.json
```

## Usage

```tsx
import { Input } from "@/components/ui/input"

<Input type="email" placeholder="Email" />
```

Supports every native input type including `file`. Invalid state styling via
`aria-invalid`.
