## Usage

```tsx
import { Kbd, KbdGroup } from "@/components/ui/kbd"

<KbdGroup>
  <Kbd>⌘</Kbd>
  <Kbd>K</Kbd>
</KbdGroup>
```

## API

| Prop | Type | Default |
| --- | --- | --- |
| `className` (Kbd / KbdGroup) | `string` | — |

Both render plain elements (`Kbd` a `<kbd>`, `KbdGroup` a `<div>`) and forward
all native attributes. Use `KbdGroup` to lay out a multi-key combination.
