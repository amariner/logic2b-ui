## Usage

```tsx
import { MotionFade } from "@/components/ui/motion-fade"

<MotionFade duration={600}>
  <Card>…</Card>
</MotionFade>
```

`MotionFade` is `<Motion preset="fade">` — see [Motion](/docs/components/motion)
for the shared `duration`, `delay`, `hover` and `asChild` props, exit recipes
and reduced-motion behavior.

## API

| Prop | Type | Default |
| --- | --- | --- |
| `duration` | `number` (ms) | `500` |
| `delay` | `number` (ms) | `0` |
| `hover` | `lift \| sink \| scale \| glow` | — |
| `asChild` | `boolean` | `false` |
| ...props | `React.ComponentProps<"div">` | — |
