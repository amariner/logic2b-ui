## Usage

```tsx
import { MotionScale } from "@/components/ui/motion-scale"

<MotionScale hover="lift">
  <Card>…</Card>
</MotionScale>
```

`MotionScale` is `<Motion preset="scale">` — see
[Motion](/docs/components/motion) for the shared `duration`, `delay`, `hover`
and `asChild` props, exit recipes and reduced-motion behavior.

## API

| Prop | Type | Default |
| --- | --- | --- |
| `duration` | `number` (ms) | `500` |
| `delay` | `number` (ms) | `0` |
| `hover` | `lift \| sink \| scale \| glow` | — |
| `asChild` | `boolean` | `false` |
| ...props | `React.ComponentProps<"div">` | — |
