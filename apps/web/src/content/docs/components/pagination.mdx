## Usage

```tsx
import {
  Pagination,
  PaginationContent,
  PaginationItem,
  PaginationLink,
  PaginationNext,
  PaginationPrevious,
} from "@/components/ui/pagination"

<Pagination>
  <PaginationContent>
    <PaginationItem>
      <PaginationPrevious href="#" />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#">1</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationNext href="#" />
    </PaginationItem>
  </PaginationContent>
</Pagination>
```

## API

`Pagination` is a set of styled elements — the parts are plain `nav`, `ul`,
`li`, and `a` wrappers, so they accept the usual HTML attributes (`href`,
`onClick`, etc.).

| Prop | Type | Default |
| --- | --- | --- |
| `isActive` (PaginationLink) | `boolean` — marks the current page (`aria-current="page"`) | `false` |
| `size` (PaginationLink) | `default \| sm \| lg \| icon` | `icon` |
