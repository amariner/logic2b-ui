## Usage

```tsx
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "@/components/ui/select"

<Select>
  <SelectTrigger className="w-[180px]">
    <SelectValue placeholder="Select a fruit" />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="apple">Apple</SelectItem>
  </SelectContent>
</Select>
```

## API

| Prop | Type | Default |

| `value` / `defaultValue` (Select) | `string` | — |

| `disabled` (Select) | `boolean` | `false` |
| `size` (SelectTrigger) | `sm \| default` | `default` |

## Accessibility

Built on [Radix Select](https://www.radix-ui.com/primitives/docs/components/select),
following the WAI-ARIA listbox pattern.

- Fully keyboard operable: open with <kbd>Enter</kbd>/<kbd>Space</kbd>/arrow
  keys, move with arrows, and type-ahead to jump to an option.
- <kbd>Esc</kbd> closes the list and focus returns to the trigger.
- The trigger exposes the selected value to assistive technology; use
  `SelectValue`'s `placeholder` for the empty state.
- Group related options with `SelectGroup` + `SelectLabel` for clearer
  announcements.
