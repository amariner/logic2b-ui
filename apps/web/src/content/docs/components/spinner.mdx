## Usage

```tsx
import { Spinner } from "@/components/ui/spinner"

<Spinner />
<Spinner className="size-6 text-primary" />
```

## API

| Prop | Type | Default |
| --- | --- | --- |
| `className` | `string` | — |

Renders lucide-react's `Loader2Icon` with `animate-spin`; any prop accepted by an
`<svg>` element (e.g. `className` for size/color) is passed through. It ships with
`role="status"` and `aria-label="Loading"` so assistive technology announces it.
